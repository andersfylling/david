Detailed Description

This class instance lives through the whole engine lifetime.

Used to generate nodes from a gameState node.

Number of levels to be generated or nr of nodes can be specified.

This class starts always on the leftmost node, generates children and then moves to the neighbour node to his right. If there are no neighbour node at this level, it's either skipped if the parent discovered chess mate or the generating is finished.
